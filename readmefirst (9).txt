This release includes the ixl FreeBSD Base Driver for Intel(R) Ethernet Network
Connections.

The ixl driver supports devices based on the following controllers:
  * Intel(R) Ethernet Controller I710
  * Intel(R) Ethernet Controller X710
  * Intel(R) Ethernet Controller XL710
  * Intel(R) Ethernet Network Connection X722
  * Intel(R) Ethernet Controller XXV710
  * Intel(R) Ethernet Controller V710

NOTE: You can download Ethernet drivers at:
https://downloadcenter.intel.com
