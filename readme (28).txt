PROAVF Winx64 NDIS68 README
***************************

The driver package files in this folder can be used to install drivers
for Intel(R) Ethernet Adaptive Virtual Function (Intel(R) Ethernet
AVF) on the following guest partition operating systems:

* Microsoft* Windows Server* 2019

The driver package supports devices based on the following
controllers:

* Intel(R) Ethernet Adaptive Virtual Function

* Intel(R) Ethernet Controller E810-C

* Intel(R) Ethernet Controller E810-XXV

* Intel(R) Ethernet Connection E823-C

* Intel(R) Ethernet Connection E823-L

* Intel(R) Ethernet Controller I710

* Intel(R) Ethernet Controller X710

* Intel(R) Ethernet Controller XL710

* Intel(R) Ethernet Network Connection X722

* Intel(R) Ethernet Controller XXV710

* Intel(R) Ethernet Controller V710
