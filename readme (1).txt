PRO40GB Winx64 Windows Server 2022 README
*****************************************

The driver package files in this folder can be used to install drivers
for Intel(R) Ethernet 700 Series devices on the following operating
systems:

* Microsoft* Windows Server* 2022

* Microsoft Azure Stack HCI

The driver package supports devices based on the following
controllers:

* Intel(R) Ethernet Controller I710

* Intel(R) Ethernet Controller X710

* Intel(R) Ethernet Controller XL710

* Intel(R) Ethernet Network Connection X722

* Intel(R) Ethernet Controller XXV710

* Intel(R) Ethernet Controller V710
